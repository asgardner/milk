## Installation

In the root directory ie. ```processtools```

```bash
pip install .
```

Note: You can install it in a custom path with the option ```-t $HOME/local/python```

Note2: If you update it, use the option ```-U```

## Usage

```python
import CacaoProcessTools as CPT
```

## Open streamCTRL
